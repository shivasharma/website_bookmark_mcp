Chrome, Firefox, Safari & Edge can all export bookmarks as a .html file. This is the fastest way to migrate — parse happens 100% in your browser, nothing is uploaded to a server.
1
Chrome: Menu → Bookmarks → Bookmark Manager → ⋮ → Export bookmarks
2
Firefox: Menu → Bookmarks → Manage Bookmarks → Import & Backup → Export Bookmarks to HTML
3
Safari: File → Export Bookmarks… → Save as HTML
4
Edge: Menu → Favorites → ⋮ → Export favorites

1

2
Inoreader: Preferences → Import/Export → Export subscriptions
3
NewsBlur: Account → Export → Download OPML

Use the REST API to save bookmarks programmatically from any script, app, or browser extension. Authenticate with your API key.
Endpoints
POSThttps://api.markd.app/v1/bookmarks
GEThttps://api.markd.app/v1/bookmarks
DELhttps://api.markd.app/v1/bookmarks/:id
Example — Save a Bookmark
curl -X POST https://api.markd.app/v1/bookmarks \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "url": "https://example.com", "title": "Example Site", "tags": ["dev", "tools"], "notes": "Saved via API" }'

1

2
Android Chrome: Menu → Add to Home screen → Install
3
Now open any link in any app, tap Share → BMark
4
BMark appears as "Save to BMark" in your native share sheet
manifest.json — Share Target Config
